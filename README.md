# Capstone 3: EmporiUm Regional Sales Report — Northeast Region
Sales performance analysis for the EmporiUm Northeast 
region using Power BI. The report covers the store and online 
sales trends, top-selling books, store performance, and 
product category insights across 4 states from January 
2022 through December 2025.

* `Ramirez_capstone3.pbix` — full Power BI report 
  covering all core requirements and bonus analysis

## Data Files Required
Place all of these in the same folder as the report 
before opening:

* `Capstone3_Sample_Sales.xlsx`
* `book_list.txt`

## What the Report Covers
1. Sales performance overview — total revenue trends 
   across the Northeast region from 2022 to 2025
2. Sales by category — revenue breakdown across books, 
   textbooks, tech gear, and other product categories
3. Revenue by state — relative sales share across Maine, 
   Maryland, Massachusetts, and New Jersey
4. Top-selling books — best performing general audience 
   books ranked by revenue, excluding textbooks
5. Store performance — top and lowest performing store 
   locations ranked by total revenue
6. Store trends — sales growth over time by store 
   location and state

## How to Open
1. Make sure all data files are in the same folder 
   as the report
2. Open `Ramirez_capstone3.pbix` in Power BI Desktop
3. If prompted, update the data source path to match 
   your local folder location
4. Click Refresh to load the latest data
